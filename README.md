# Pizza Customizer

## Run

1. `npm install`
2. `npm run dev`

## Tests

1. `npm run test`

## Lint and format

1. `npm run lint`
2. `npm run format`

## Description

Pizza order manager with CRUD, theme persistence (light/dark/system), and live price.

- React (Vite) + TypeScript
- Tailwind CSS
- Zustand
- React Hook Form + Zod

## Structure

- `src/components` — UI kit and layout
- `src/features` — pizza features
- `src/store` — Zustand stores
- `src/utils` — utilities
- `src/types` — types
- `src/__tests__` — tests
